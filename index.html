HTML - язык разметки страниц Интернет

Задания для лабораторных занятий по освоению HTML и CSS

 

Пояснительная записка

Вам предлагается набор заданий, последовательное аккуратное и вдумчивое выполнение которых позволит разобраться с азами HTML и CSS, получить опыт создания простых, но вполне состоявшихся гипертекстовых документов Интернет.

Каждое задание - это точный пример того, что вы должны сделать. Текст и графику для создаваемых вами страниц вы можете брать с этих же примеров. Создавать страницы - в простом текстовом редакторе.

Мы понимаем, что не составляет никакого труда скопировать и весь пример целиком, однако в этом случае вы лишите себя главного - опыта по созданию страниц Интренет. Мы рекомендуем вам в каждом случае выполнять задание "без подглядывания" в исходные тексты примеров, обращаясь к ним лишь для сравнения результатов, после того, как ваш вариант страницы будет окончательно завершен.

Минимальный набор необходимого справочного материала, который вам потребуется при создании ваших страниц, приведен в приложениях. Приветствуется также и использование дополнительной литературы из Интернет, которую вы сможете найти сами.

Желаем удачи в нелегком труде по созданию сайтов!
 

Задания

  • Задание 1, выполнив которое, вы познакомитесь с основными тегами HTML для оформления простых текстов
     
  • Задание 2, в котором от вас потребуется умение включать в документы HTML графические изображения, правильно настраивать способы сопряжения изображений и текста
     
  • Задание 3, где потребуется составить несложную табличку, для чего и само содержание сказки придется несколько поменять
     
  • Задание 4, в ходе выполнения которого вы научитесь оформлять не только относительные ссылки на другие документы HTML (которые придется дополнительно создать), но и абсолютные - на существующие сайты в Интернет
     
  • Задание 5, где создается несложная панель навигации по документу, для чего потребуется не только расставить метки в нужных местах страницы, но и использовать таблицы для создания новой структуры HTML-документа
     
  • Задание 6, в котором создается оформление внешнего вида документа на основе каскадных таблиц стилей, в результате чего некоторые элементы страницы получают новый облик, недоступный при использовании стандартных средств оформления HTML
     
  • Задание 7 (повышенной сложности), при выполнении которого для достижения простой и понятной цели придется проявить не только знания, но и смекалку
 

Приложения

  • Справка по тегам HTML
  • Безопасные цвета Интернет
  • Руководство по CSS
